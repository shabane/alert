# alert

> a simple notifyer which show the title and message that you specify in the terminal

---

> __how to install__
>
>```bash
>git clone https://github.com/shabane/alert.git && cd alert
>sudo ./install.sh
>```



> __Usage__:
>
>  ```alert 'title' 'message'```
